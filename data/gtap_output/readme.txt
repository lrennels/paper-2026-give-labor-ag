Country - 160 GTAP regions (see https://www.gtap.agecon.purdue.edu/databases/regions.aspx?Version=11.211)
Name - GTAP Region Long Name           
Region - Geographic Region 
Scenario – “Crops Only” and “Labor only (All Sectors) “
Degrees - 1 to 4 by 0.5 increments relative to pre-industrial
Percentile - 2.5, 50.0 and 97.5   (for Crops Only scenario)
GCM -   15 GCM and 1 model_ensemble   (for Labor only (All Sectors))
Welfare_in_M_USD - Changes in Welfare in million USD
Valueadded_in_M_USD - Changes in Value Added in million USD (as is)

Instructions for calculating % welfare changes:
For the “Labor (All Sector)” scenario, use the values in Initial_Income_in_M_USD to calculate the % change in welfare
For the “Crops Only” scenario, use the values in Valueadded_in_M_USD to calculate the % change in welfare
